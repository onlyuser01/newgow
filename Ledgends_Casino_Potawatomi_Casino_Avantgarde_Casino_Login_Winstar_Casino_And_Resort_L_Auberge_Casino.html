A Cluster Pays Game Packed with Features

The slot is played on a 5×7 grid and it can be smoothly enjoyed on all compatible mobile devices in portrait mode.

The space guardian theme is nicely complemented by the one-of-a-kind selection of game symbols chosen by the talented Swintt team. The list ranges from the specific elements of wind, water, fire, and earth, and it also includes the celestial beings that represent them.

Since it is a cluster pays slot, players will notice the missing paylines. Instead, gamers need to focus on landing clusters of adjacent symbols. When this occurs, the respective symbols are seen exploding and disappearing from the grid, making room for fresh symbols to drop down to their position.

The cluster pays mechanic can be easily triggered with the help of the substitute wilds as well as the four available elemental boosters which players can use during each new spin. Elemental boosters are engaged by landing symbol combinations that feature a minimum of 25 identical icons that can fill up the element meter during just one spin.

Since the cluster pays mechanic is a continuous one, the identical symbols do not need to be a part of a single win. Instead, they can be spread across several wins so the meter can be filled more easily.

Every elemental booster is responsible for applying a certain bonus effect to the grid. Players can also simultaneously engage multiple boosters once all four element meters are filled up.
